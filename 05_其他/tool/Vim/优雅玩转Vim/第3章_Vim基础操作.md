# 第3章 Vim基础操作

## 3.1 移动、跳转、缩进

### 移动

> 普通模式下按如下键

+ `h`:左移
+ `l`:右移
+ `j`:下移
+ `k`:上移
+ `$`:移动到行尾

### 3-3 修改 查找 替换

> 按ESC后执行如下命令

+ **行号+gg:跳转到指定行**
+ i从当前光标处插入
+ I从所在行的第一个非空字符出插入
+ a从光标之后插入，A从行尾插入
+ **o从下一行插入，O从上一行插入**
+ r替换当前字符
+ R替换字符直到ESC输入
