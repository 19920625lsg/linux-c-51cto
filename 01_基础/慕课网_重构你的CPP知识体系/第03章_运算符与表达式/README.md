## 3-10 补码计算
+ 对正数：直接按位计算权重和
  > `0101: 2^2+2^0 = 5`
+ 对负数：保留符号位(0为正1为负)，对后面每位取反加1
  > `1011: -(~011)  = -(100) = -5`
